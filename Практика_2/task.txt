1.	Реализуйте программу, которая принимает на вход список строк и возвращает новый список, в котором каждая строка разбита на отдельные символы, а затем объединена в одну строку через пробел.
2.	Напишите программу, которая принимает на вход два словаря и определяет, содержат ли они разные ключи, используя операторы сравнения.
3.	Напишите программу, которая принимает две строки и использует хэш-функции для определения, является ли первая строка анаграммой второй строки, без явного сравнения символов
4.	Напишите программу, которая считывает строку, содержащую числа, и возвращает список чисел, отсортированный в порядке убывания.
5.	Напишите программу, которая считывает два числа в виде строки и возвращает их НОД (наибольший общий делитель) с использованием алгоритма Эвклида, а также выводит количество шагов, выполненных алгоритмом.
